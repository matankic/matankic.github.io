Learning to See Inside Opaque Liquid Containers using Speckle Vibrometry

Matan Kichler, Shai Bagon, Mark Sheinin
Weizmann Institute of Science
ICCV 2025
Paper Supp Video Code Data Model BibTeX

We present a novel speckle-based vibrometry system that captures surface vibrations of opaque liquid containers remotely, enabling first-of-its-kind inference of hidden liquid levels without physical handling or weighing.
