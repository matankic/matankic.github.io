Containers Dataset

This list below details the different container types used in our experiments. Each container has a uniquely interesting response to the speaker's excitation, which you can play by clicking underneath the container name (using headphones is recommended). The container's walls also generate (or pass) sound inside the container, depending on which frequencies are attenuated and which are not. Thus, the sound you hear approximates the sound `heard' within the container (e.g., by a small ant).
